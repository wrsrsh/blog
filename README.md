## blog

This is my blog at [reshi.me](https://reshi.me) built with [Astro](https://astro.build) and hosted on [cloudflare workers](https://workers.cloudflare.com/)

---

I write this blog in [Obsidian](https://https://obsidian.md/) as `mdx` and then use [Warblog](https://github.com/wrsrsh/warblog) to just publish it from there.

Have fun!
